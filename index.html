3. Add it to Outlook

  1. Classic Outlook (desktop): File › Options › Mail › Signatures › New › name it PartsCheck › click inside the editing box › press Ctrl+V. If a small paste icon appears, choose Keep Source Formatting.
  2. New Outlook / Outlook on the web: Settings (gear) › Mail › Compose and reply (or Accounts › Signatures) › New signature › click inside the box › Ctrl+V.
  3. Outlook for Mac: Outlook › Settings › Signatures › + › click inside the box › Cmd+V.
  4. Set it as the default for New messages and Replies/forwards, then Save.
  5. Send yourself a test email and check it looks right. Never retype or edit text inside Outlook's signature box — come back here instead.
